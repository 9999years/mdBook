mdBook Documentation

Rust Library

Check here for the API docs generated by rustdoc mdBook is not only a command line tool, it can be used as a crate. You can extend it, integrate it in current projects. Here is a short example:

extern crate mdBook;

use mdBook::MDBook;

fn main() {
    let book =  MDBook::new("my-book")  // Path to root
                    .set_src("src")     // Path from root to source directory
                    .set_dest("book")   // Path from root to output directory
                    .read_config()      // Parse book.json file for configuration

    book.build().unwrap();              // Render the book
}
